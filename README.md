# Advanced Math Data Science Exam Preparation

## 概要
このリポジトリは、大学で学習する数学とデータサイエンスに関する演習問題の解答例とコードを管理するためのものです。6章にわたる演習問題を体系的に整理し、各問題に対する解答コードや説明資料を提供します。
